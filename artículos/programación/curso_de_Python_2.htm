Instalación de Python

Para programar en Python es imprescindible contar con su intérprete, el cual puedes (y debes) descargar en la página oficial de PythonTM. El enlace anterior lleva directamente a la página de descargas. Allí solo tienes que elegir tu sistema operativo y la versión que quieres instalar.

Te recomiendo que instales la versión estable más reciente posible, aunque considero eso una opción personal. Yo trabajaré en esta pequeña guía con la última versión disponible a esta fecha (29 de noviembre del 2015): Python 3.

Creando nuestro primer programa: «funciona.py»

Una vez descargado el interprete de Python, lo que debemos hacer es comprobar si funciona correctamente. Para ello crearemos nuestro primer programa: print("Funciona"). La función print muestra la información especificada entre paréntesis por pantalla. Si queremos mostrar un mensaje, que es una cadena de texto (tranquilo, hablaremos de esto más adelante) debemos escribirlo entre comillas simples ('...') o dobles ("..."). Es indiferente qué comillas utilicemos, pero no podemos mezclarlas, si lo hacemos saltará el siguiente error: SyntaxError: EOL while scanning string literal.

Este programa lo ejecutaremos en el intérprete. Si el programa muestra por pantalla «funciona», Python se ha instalado correctamente, o al menos podemos usar gran parte de sus funciones.

Guardando el programa

El programa que acabamos de crear lo hemos escrito en el intérprete y luego lo hemos ejecutado pulsando «Enter»; pero para poder ejecutar un programa cuando queramos y no tener que escribirlo constantemente en el intérprete es preferible guardarlo.

Para guardar un programa primero hay que escribirlo. Si queremos guardar el programa, no lo escribiremos en el intérprete, sino en un editor de texto. No sirve cualquier editor; no puede dar formato al texto. LibreOffice Writter ni Microsoft Word sirven. Podemos usar un editor simple, como el Bloc de Notas de Windows; un editor más específico para programadores, como Sublime Text; o un IDE, como Visual Studio o NetBeans
